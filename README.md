# djdev-workshop

Claude Code plugin marketplace by [djimontyp](https://github.com/djimontyp).

## Plugins

| Plugin | Description | Category |
|--------|-------------|----------|
| [claude-session-digest](./plugins/claude-session-digest/) | Auto-summarize Claude Code sessions into daily markdown digests | productivity |

## Installation

```bash
# Add to your Claude Code settings
# Settings → Extra Known Marketplaces → add this repo URL
```

Or install manually:

```bash
claude plugin install djimontyp/djdev-workshop/claude-session-digest
```
